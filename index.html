Tutorial de Uso da Planilha IMPORTAÇÃO

Guia completo para automatizar a importação de dados financeiros para o sistema Contimatic

Este tutorial foi desenvolvido para auxiliar os usuários da planilha "IMPORTAÇÃO", uma ferramenta criada para automatizar a importação de dados financeiros para o sistema Contimatic. O documento apresenta instruções passo a passo sobre como utilizar as principais funcionalidades da planilha.

Requisitos

  • Microsoft Excel com suporte a macros (arquivo .xlsm)
  • Acesso ao sistema Contimatic (para importação completa)
  • Permissões adequadas para as pastas de destino dos arquivos gerados

Como usar este tutorial

Este tutorial está organizado em seções que cobrem desde os primeiros passos até a solução de problemas comuns. Utilize o menu de navegação para acessar diretamente a seção que você precisa consultar.

Primeiros Passos

Aprenda como começar a utilizar a planilha IMPORTAÇÃO

Abrindo a Planilha

  1. Abra o arquivo "3- IMPORTAÇÃO .xlsm"
  2. Se aparecer um aviso de segurança sobre macros, clique em "Habilitar Conteúdo"
  3. Ao abrir a planilha pela primeira vez, será exibida uma mensagem perguntando se você deseja ver o tutorial em vídeo
    • Clique em "Sim" para abrir o vídeo tutorial no YouTube
    • Clique em "Não" para continuar sem assistir ao vídeo

Dica Importante

É recomendável assistir ao vídeo tutorial na primeira vez que utilizar a planilha. Ele fornece uma visão geral das funcionalidades e pode ajudar a compreender melhor o fluxo de trabalho.

Interface Principal

A planilha possui vários botões na aba "COLA" que controlam as principais funcionalidades:

INDIVIDUAL

Seleciona o modo de importação individual

MULTIPLO

Seleciona o modo de importação múltipla

TOTAL

Ativa/desativa o modo de importação completa automatizada

LIMPAR

Limpa os dados da planilha para uma nova importação

TUTORIAL

Abre o vídeo tutorial no YouTube

GERAR

Inicia o processo de importação

ADMINISTRADOR

Ativa/desativa o modo administrador (requer senha)

Processo de Importação

Guia passo a passo para realizar importações de dados

Passo 1: Preparar os Dados

  1. Na aba "COLA", insira os dados financeiros a serem importados
    • Coluna C: Data de vencimento
    • Coluna D: Descrição
    • Coluna E: Número da NF
    • Coluna F: Valor do documento
    • Coluna G: Acréscimos
    • Coluna H: Juros
    • Coluna I: Descontos
    • Coluna J: Valor líquido
    • Coluna K: Conta de pagamento
    • Coluna L: Loja
  2. Certifique-se de que todos os dados estão preenchidos corretamente, especialmente:
    • Datas no formato correto
    • Valores numéricos nas colunas de valores
    • Descrições dos fornecedores existentes no cadastro

Passo 2: Selecionar o Tipo de Importação

Escolha entre os modos de importação disponíveis:

INDIVIDUAL

Clique no botão "INDIVIDUAL" para processar cada lançamento separadamente

  • Recomendado quando há juros, descontos ou tarifas a serem tratados individualmente
  • Permite maior controle sobre cada lançamento

MULTIPLO

Clique no botão "MULTIPLO" para processar lançamentos em grupo

  • Mais eficiente para importações em lote
  • Agrupa lançamentos relacionados

Passo 3: Opção de Importação Completa (Opcional)

Se desejar que o sistema realize a importação completa automaticamente:

  1. Clique no botão "TOTAL" (o botão ficará com transparência parcial quando ativado)
  2. Na primeira vez que usar esta função, será exibido um aviso sobre possíveis erros
  3. Esta opção automatiza todo o processo até o sistema Contimatic

Observação Importante

A importação completa depende da configuração correta do sistema Contimatic e pode estar sujeita a erros se houver mudanças no sistema.

Passo 4: Iniciar o Processo

  1. Clique no botão "GERAR" para iniciar o processo de importação
  2. O sistema realizará as seguintes operações:
    • Verificação de linhas vazias
    • Ajuste de formatos de dados
    • Verificação de fornecedores cadastrados
    • Preparação dos dados para importação
    • Geração dos lançamentos na aba "IMPORT"
  3. Se houver fornecedores não cadastrados:
    • Será exibida uma mensagem informando que existem fornecedores para cadastrar
    • Você será direcionado para a aba "CADASTRO" para incluir os novos fornecedores
  4. Se todos os fornecedores estiverem cadastrados e o modo "TOTAL" estiver ativado:
    • O sistema prosseguirá com a importação completa para o Contimatic
    • Será criado ou atualizado um arquivo Excel na pasta do sistema Contimatic
  5. Ao final do processo, será exibida uma mensagem de confirmação ":)"

Funcionalidades Adicionais

Explore recursos extras da planilha IMPORTAÇÃO

Limpeza de Dados

Para limpar os dados e iniciar uma nova importação:

  1. Clique no botão "LIMPAR"
  2. Todos os dados nas abas "COLA", "CONCILIACAO" e "IMPORT" serão limpos
  3. Os cabeçalhos e configurações serão mantidos

Controle de Numeração

Para alterar o número do próximo lançamento:

  1. Clique na célula M2 na aba "COLA"
  2. Digite o número desejado para o próximo lançamento
  3. O sistema atualizará automaticamente a numeração para os próximos lançamentos

Modo Administrador

Para ativar o modo administrador (permite edição de áreas protegidas):

  1. Clique no botão "ADMINISTRADOR"
  2. Digite a senha de administrador quando solicitado
  3. Quando o modo administrador estiver ativo, o botão ficará com transparência parcial
  4. Para desativar o modo administrador, clique novamente no botão

Atenção

O modo administrador deve ser utilizado com cautela, pois permite a edição de áreas protegidas da planilha que podem afetar o funcionamento da ferramenta.

Cadastro de Fornecedores

Se for necessário cadastrar novos fornecedores:

  1. Acesse a aba "CADASTRO"
  2. Localize a primeira linha vazia na coluna B
  3. Insira o nome do fornecedor na coluna B
  4. Preencha o código do fornecedor na coluna C
  5. Se necessário, preencha as informações adicionais nas demais colunas

Reiniciar a Ferramenta

Se ocorrerem problemas ou para redefinir as configurações:

  1. Clique com o botão direito em qualquer área da planilha
  2. Selecione "Reset" no menu de contexto
  3. Será perguntado se deseja manter as anotações
  4. A ferramenta será reiniciada com as configurações padrão

Solução de Problemas Comuns

Encontre soluções para problemas frequentes durante o uso da planilha

Dicas e Boas Práticas

Verifique os dados antes de importar

  • Certifique-se de que todas as datas estão no formato correto
  • Confirme se os valores estão corretos e nas colunas apropriadas
  • Verifique se todos os fornecedores estão cadastrados

Backup regular

  • Faça cópias de segurança da planilha regularmente
  • Mantenha versões anteriores em caso de necessidade de recuperação

Tratamento de erros

  • Se ocorrerem erros durante a importação, verifique:
  • Formato dos dados
  • Cadastro de fornecedores
  • Permissões de acesso às pastas do sistema

Importação em lotes menores

  • Para grandes volumes de dados, considere dividir em lotes menores
  • Isso facilita a identificação e correção de possíveis erros

Modo de importação adequado

  • Use o modo MULTIPLO para lançamentos com juros, descontos ou tarifas
  • Use o modo INDIVIDUAL para lançamentos simples em lote

Problemas Específicos

Mensagem "Planilha vazia!"

  • Verifique se há dados inseridos na aba "COLA" a partir da linha 2
  • Certifique-se de que a coluna C (data de vencimento) está preenchida

Mensagem "Selecione o Layout de Importação!"

  • Clique em um dos botões de modo de importação (INDIVIDUAL ou MULTIPLO)

Mensagem "Existem fornecedores para serem cadastrados!"

  • Acesse a aba "CADASTRO"
  • Cadastre os fornecedores indicados
  • Retorne à aba "COLA" e tente novamente

Planilha travada ou protegida

  • Se você precisar editar áreas protegidas, ative o modo administrador
  • Se não tiver a senha de administrador, contate o responsável pela planilha

Erro na importação completa

  • Verifique se o caminho para a pasta do Contimatic está correto
  • Confirme se você tem permissões de acesso à pasta
  • Verifique se o arquivo de destino não está aberto em outro programa

Conclusão

Esta planilha foi desenvolvida para facilitar o processo de importação de dados financeiros para o sistema Contimatic. Seguindo este tutorial, você poderá utilizar todas as funcionalidades disponíveis de forma eficiente e segura.

Para dúvidas adicionais ou suporte técnico, consulte o administrador responsável pela planilha ou assista ao vídeo tutorial disponível através do botão "TUTORIAL".
